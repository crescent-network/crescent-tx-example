# Crescent Transaction Example

This repository is a sample project to demonstrate how to wrap message(s) in a single transaction.

## Dependency

| Dependency    | Version |
| ------------- | ------- |
| Go            | 1.18    |
| Crescent Core | 2.1.1   |

## Usage

```bash
# See the code in main.go file for more in detail.
go run main.go
```
